## 1. АРИФМЕТИКА МОДЕЛИ

### Операционные расходы

```
Аренда/год     = 275 м² × 2 500 руб. × 12 мес. / 1 000 000 = 8,250 млн
ФОТ/мес.       = 50 чел. × 94 800 руб. = 4,740 млн
ФОТ+взносы/мес.= 4,740 × 1,30 = 6,162 млн
ФОТ+взносы/год = 6,162 × 12 = 73,944 млн
Коммунальные   = 0,396 млн/год
Прочие (8% ФОТ)= 73,944 × 0,08 = 5,916 млн (ВНИМАНИЕ: 8% от ФОТ с взносами или без?)
               Если от ФОТ без взносов: 4,740 × 12 × 0,08 = 4,550 млн
               Если от ФОТ с взносами: 73,944 × 0,08 = 5,916 млн ← модель использует это

ИТОГО/год      = 8,250 + 73,944 + 0,396 + 5,916 = 88,506 млн ✓
ИТОГО/мес.     = 88,506 / 12 = 7,376 млн (модель: 7,375 — расхождение < 0,02%, ОК)
ИТОГО/кв.      = 88,506 / 4 = 22,127 млн (модель: 22,126 — расхождение < 0,005%, ОК)
```

### Денежные потоки по модели (Сценарий А)

```
Кварталы 1–4 (инвестиции):
  Поток/кв.  = −312,500 − 22,126 = −334,626 млн
  Кумулятив кв.4 = −334,626 × 4 = −1 338,504 млн

Кварталы 5–8 (первые доходы):
  Поток/кв.  = 356,750 − 22,126 = 334,624 млн
  Кумулятив кв.8 = −1 338,504 + 334,624 × 4 = −1 338,504 + 1 338,496 = −0,008 млн
  ≈ 0 (окупаемость на конец кв. 8) ✓

Кварталы 9–12 (рост):
  Поток/кв.  = 522,130 − 22,126 = 500,004 млн
  Кумулятив кв.12 = −0,008 + 500,004 × 4 = −0,008 + 2 000,016 = +2 000,008 млн
  ≈ +2 000 млн ✓
```

### Проверка: лист Модель vs лист PnL

```
PnL: инвестиции = 1 250 млн (только инвестор)
Модель: общий бюджет = 1 850 млн (1 250 + 600 прав)

PnL: окупаемость = кв. 8 (24 мес.) — считает ТОЛЬКО денежные инвестиции 1 250
Модель: окупаемость = 36–48 мес. — считает ОБЩИЙ бюджет 1 850

Разрыв = 12–24 мес. → причина: разная база расчёта
```

## 5. ТОЧКА НЕВОЗВРАТА

```
Потеря > 50% = > 625 млн
Расходы инвестора = 1 250 (инвестиции) + 88,5 × 3 (опер.) = 1 515,5 (без маркетинга)
                  = 1 250 + 265,5 + 500 (маркетинг) + 125 (непредвиденные) = 2 140,5 (с маркетингом)

Доля инвестора в доходах = 60% (до окупаемости)

Без маркетинга:  1 250 − 0,6 × Доходы > 625 → Доходы < 1 042 млн
С маркетингом:   (1 250 + 500 + 125) − 0,6 × Доходы > 625 → Доходы < 2 083 млн
                 Т.е. если общие доходы < 2 083 млн (менее 46% от плана) — потеря > 50%
```
