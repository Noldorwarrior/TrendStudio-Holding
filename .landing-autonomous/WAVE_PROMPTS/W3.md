## Код Team 2-state

```jsx
const [expandedId, setExpandedId] = useState(null);

<article 
  role="button" 
  aria-expanded={expandedId === m.id}
  tabIndex={0}
  onClick={() => setExpandedId(expandedId === m.id ? null : m.id)}
  onKeyDown={e => (e.key==='Enter' || e.key===' ') && setExpandedId(m.id)}
  style={{
    position: 'relative',
    transform: expandedId === m.id ? 'scale(1.15)' : (expandedId ? 'scale(0.92)' : 'scale(1)'),
    opacity: expandedId && expandedId !== m.id ? 0.5 : 1,
    zIndex: expandedId === m.id ? 100 : 1,
    transition: 'all 400ms cubic-bezier(0.22, 1, 0.36, 1)',
    background: 'linear-gradient(135deg, #F4A261, #2A9D8F) padding-box, linear-gradient(135deg, #F4A261, #2A9D8F) border-box',
    border: '3px solid transparent',
    backgroundClip: 'padding-box, border-box',
    willChange: 'transform',
    transformOrigin: 'center'
  }}
>
  <img src={`__IMG_PLACEHOLDER_${m.id}__`} alt={m.alt}/>
  <div>{m.slot}</div>
  {expandedId === m.id && <div className="expanded">{m.name} — {m.bio}</div>}
</article>
```

## Pipeline card правильный 3D-tilt

```jsx
const handleMouseMove = (e) => {
  const rect = e.currentTarget.getBoundingClientRect();
  const x = (e.clientX - rect.left) / rect.width - 0.5;
  const y = (e.clientY - rect.top) / rect.height - 0.5;
  e.currentTarget.style.transform = `perspective(1000px) rotateY(${x*5}deg) rotateX(${-y*5}deg) scale(1.02)`;
};
const handleLeave = (e) => {
  e.currentTarget.style.transform = '';
};

<article 
  onMouseMove={handleMouseMove} 
  onMouseLeave={handleLeave}
  style={{
    transformOrigin: 'center center',
    willChange: 'transform',
    transition: 'transform 400ms cubic-bezier(0.22, 1, 0.36, 1)'
  }}
>
  <!-- card content -->
</article>

<!-- parent container: -->
<div style={{perspective: 1000}}>
```
