### 3.1 Donut chart «100% revenue-mix»

```jsx
import { PieChart, Pie, Cell, Tooltip as RechartsTooltip, ResponsiveContainer } from 'recharts';

const CHANNELS = [
  { id:'theatrical', name:'Театральный прокат', pct:30, color:'#F4A261', window_months:3, partners:['Central Partnership-like','Каропрокат-like'] },
  { id:'ott',        name:'OTT / Streaming',    pct:40, color:'#2A9D8F', window_months:12, partners:['Кинопоиск','Okko','Wink','START'] },
  { id:'tv',         name:'TV',                 pct:10, color:'#4A9EFF', window_months:24, partners:['Первый канал','НТВ','СТС'] },
  { id:'educational',name:'Educational / B2B',  pct:5,  color:'#EAB308', window_months:36, partners:['ВГИК','МШК','Netology'] },
  { id:'international',name:'International sales',pct:15,color:'#EF4444', window_months:0, partners:['Sales agents (СНГ/Азия/BRICS)','Netflix (selective)'] }
];

function DistributionDonut({ hoverChannel, setHoverChannel }) {
  return (
    <ResponsiveContainer width="100%" height={320}>
      <PieChart>
        <Pie
          data={CHANNELS}
          dataKey="pct"
          cx="50%" cy="50%"
          innerRadius={70} outerRadius={120}
          paddingAngle={2}
          onMouseEnter={(_, idx) => setHoverChannel(CHANNELS[idx].id)}
          onMouseLeave={() => setHoverChannel(null)}
        >
          {CHANNELS.map(c => (
            <Cell 
              key={c.id} fill={c.color}
              opacity={hoverChannel ? (hoverChannel === c.id ? 1 : 0.3) : 0.9}
              style={{cursor:'pointer', transition:'opacity 0.2s'}}
            />
          ))}
        </Pie>
        <RechartsTooltip contentStyle={{background:'#15181C', border:'1px solid #2A2D31', borderRadius:8}} />
      </PieChart>
    </ResponsiveContainer>
  );
}
```

**Персональный LP-пример под слайдером:**

```jsx
function PersonalExample({ multiplier }) {
  const [commit, setCommit] = useState(100);
  const scenarioGross = commit * multiplier;
  const scenarioProfit = scenarioGross - commit;
  const lpTake = commit + scenarioProfit * 0.80 * 0.85; // упрощённая формула
  
  return (
    <Reveal>
      <div style={{marginTop:24, padding:20, background:'#15181C', borderRadius:10, border:'1px dashed #F4A26133'}}>
        <div style={{display:'flex', gap:12, alignItems:'center', flexWrap:'wrap'}}>
          <span style={{color:'#EAEAEA'}}>На commitment</span>
          <input 
            type="number" min="10" max="500" value={commit}
            onChange={e => setCommit(+e.target.value || 10)}
            style={{width:80, padding:'4px 8px', background:'#0B0D10', border:'1px solid #2A2D31', borderRadius:6, color:'#EAEAEA'}}
          />
          <span style={{color:'#EAEAEA'}}>млн ₽ при multiplier <strong style={{color:'#F4A261'}}>{multiplier}×</strong>:</span>
        </div>
        <div style={{marginTop:12, color:'#EAEAEA', lineHeight:1.8}}>
          • Ваши деньги превратились в <strong>{scenarioGross.toFixed(0)} млн ₽</strong><br/>
          • Из них вы получаете <strong style={{color:'#2A9D8F'}}>{lpTake.toFixed(0)} млн ₽</strong> ({(lpTake/commit).toFixed(2)}×)<br/>
          • Команда получает {(scenarioGross - lpTake).toFixed(0)} млн ₽
        </div>
      </div>
    </Reveal>
  );
}
```

**Подсветка активных tier'ов:**
- При multiplier < 1.08: активен только Tier 1 → остальные opacity 0.3
- При multiplier 1.08-2.5: T1 + T2 + T3 активны, T4 opacity 0.3
- При multiplier > 2.5: все 4 tier активны
