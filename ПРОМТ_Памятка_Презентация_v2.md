# ПРОМТ: Лист-памятка спикера к инвестиционной презентации

## Роль
Ты — финансовый аналитик, готовящий спикер-карту (cheat sheet) для CEO, который будет выступать перед инвесторами. Документ будет использоваться двояко: (1) как личная шпаргалка при подготовке к встрече и (2) как памятка под рукой во время питча для быстрого ответа на вопросы.

## Контекст
- **Компания:** Тренд Студио Холдинг — контентная фабрика нового поколения (кинопроизводство + реклама + фестивали + обучение)
- **Раунд:** 1,25 млрд ₽ за 51% (стратегический инвестор)
- **Горизонт:** 3 года (2025–2028)
- **Презентация:** 6 слайдов (Титул, Рынок, Финансы, Экономика направлений, Инвестпредложение, Команда и Milestones)

## Входные данные
Используй данные из приложенных файлов:
1. **Презентация** (pptx или pdf) — все цифры и метрики со слайдов (это ФИНАЛЬНАЯ версия, цифры в ней — приоритет)
2. **Excel-финмодель** «Холдинг Кино.xlsx» — листы PnL, Расходы, Доходы, модель (используй для подтверждения расчётов)

## Выходной формат
**Word-документ (.docx)**, отформатированный для печати:
- A4, книжная ориентация
- Поля: верх/низ 2 см, лево 3 см, право 1,5 см
- Шрифт: Times New Roman 14pt, по ширине, красная строка 1,5 см, межстрочный 1,15
- Таблицы: 12pt, межстрочный 1,0
- Заголовки: жирный, синий (#0070C0) — H1 22pt, H2 18pt, H3 16pt

## Структура документа

### ЧАСТЬ 1: РАЗБОР ПО СЛАЙДАМ

Для каждого слайда (1–6) создай раздел со следующей структурой:

**Заголовок слайда** (как в презентации)

Таблица для КАЖДОЙ метрики/цифры на слайде:

| Метрика | Значение | Формула / расчёт | Источник | Ответ инвестору |
|---------|----------|-------------------|----------|-----------------|

Где:
- **Метрика** — название (как на слайде)
- **Значение** — цифра (как на слайде)
- **Формула / расчёт** — как именно считалась, с числами. Если это целевой показатель или экспертная оценка — так и напиши
- **Источник** — откуда взята: лист Excel + строка/ячейка, или «рыночные данные», или «целевой показатель»
- **Ответ инвестору** — готовая формулировка в 1–2 предложениях для уверенного ответа на «откуда эта цифра?»

### ЧАСТЬ 2: ГЛОССАРИЙ ТЕРМИНОВ И АББРЕВИАТУР

Полная расшифровка КАЖДОГО термина и аббревиатуры из презентации. Минимальный список (дополни, если в презентации есть другие):

- TAM, SAM, SOM
- CAGR
- EBITDA
- Net margin
- ROI
- LTV
- ARPU
- NPS
- COM (комиссия)
- EV/EBITDA
- Money Multiple (MM)
- IRR
- Pre-money valuation
- Breakeven
- Pipeline
- OTT
- Board seat
- DCF
- FTA / Pay TV
- SVOD / AVOD / TVOD

Для каждого термина:
1. **Аббревиатура** (англ.) → **Расшифровка** (англ.)
2. **Перевод** на русский
3. **Определение** простым языком (1–2 предложения, понятных неспециалисту)
4. **В нашем случае** — конкретный пример из презентации (как применяется)

### ЧАСТЬ 3: FAQ — КАВЕРЗНЫЕ ВОПРОСЫ ИНВЕСТОРОВ

8–12 типичных «сложных» вопросов, которые может задать инвестор, с готовыми ответами. Обязательно включить:

1. «Почему EBITDA почти равна выручке? Где расходы?»
2. «Откуда мультипликатор 7x? Не завышен ли?»
3. «MM 8,6x — это при exit. А если exit не состоится?»
4. «CAGR 115% — это реалистично? Какие аналоги на рынке?»
5. «Почему 51% за 1,25 млрд? Как считали pre-money?»
6. «Какие риски? Что если фильм провалится?»
7. «Кто конкуренты? Чем вы лучше?»
8. «Какая exit-стратегия?»

Для каждого вопроса:
- **Вопрос** (как его может задать инвестор)
- **Ключевой тезис** (1 предложение — главная мысль ответа)
- **Развёрнутый ответ** (3–5 предложений с цифрами из презентации)
- **Подкрепление** (ссылка на слайд или метрику)

## Стиль
- Лаконичный, уверенный, без воды
- Показывай только финальные (исправленные) цифры из презентации
- НЕ упоминай расхождения между файлами или внутренние нестыковки Excel — документ для спикера, а не для аудитора
- Формулировки ответов — разговорные, но профессиональные (как CEO говорит с инвестором, а не как отчёт)

## Важно
- Если цифра не выводится из Excel напрямую — честно пиши «целевой показатель» или «экспертная оценка рынка»
- Документ должен помочь спикеру УВЕРЕННО ответить на ЛЮБОЙ вопрос «откуда эта цифра?» и «что это значит?»
- Глоссарий должен быть понятен человеку без финансового образования
