## Тех-требования

- React hooks: useState, useEffect, useRef, useMemo — только
- lucide-react: Film, TrendingUp, Award, ChevronDown (и др. по нужде)
- **ЗАПРЕЩЕНО:** framer-motion, localStorage, sessionStorage, fetch, GSAP
- Все цвета через inline style (Tailwind не поддерживает arbitrary hex)
- prefers-reduced-motion: отключить count-up и bounce
- Images: **только placeholder'ы `__IMG_PLACEHOLDER_imgNN__`**, НЕ base64-инлайн сам (orchestrator инжектит)
- export default function App_W1() { ... }

## Acceptance (проверки субагента перед завершением)

1. Файл `WAVE_1_ARTIFACT.jsx` создан
2. grep: "3000" "7" "ТрендСтудио" — все найдены
3. grep: "__IMG_PLACEHOLDER_img17__" + "img19" + "img20" — все 3 placeholder'а
4. Код синтаксически валиден (babel parse)
5. Нет запрещённых API (grep localStorage/sessionStorage = 0)
