### a11y WCAG AA
- `<main>`, `<nav>`, `<header>`, `<footer>` landmarks
- Все `<img>` имеют alt (из landing_img_meta по lang)
- focus-visible outline: `*:focus-visible { outline: 2px solid #F4A261; outline-offset: 2px; }`
- Accordion/Modal: aria-expanded, aria-labelledby, aria-hidden
- prefers-reduced-motion: `@media (prefers-reduced-motion: reduce) { *,*::before,*::after { transition:none!important; animation:none!important; } }`
- Контраст: #EAEAEA на #0B0D10 = 17.52 ✅ AAA
