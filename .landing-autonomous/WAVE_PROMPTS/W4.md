## §6. M2 PIPELINE BUILDER — ПОЛНЫЙ РЕФАКТОР (конструктор портфеля)

**⚠️ УДАЛИ v1.0 логику rail+колонок-дубль.** Новая семантика:

```jsx
const PIPELINE_SEED = [
  { id:'P01', title:'Проект Alpha',   stage:'prod',    budget:350, irr:28 },
  { id:'P02', title:'Проект Bravo',   stage:'pre',     budget:280, irr:32 },
  { id:'P03', title:'Проект Charlie', stage:'pre',     budget:600, irr:26 },
  { id:'P04', title:'Проект Delta',   stage:'prod',    budget:520, irr:24 },
  { id:'P05', title:'Проект Echo',    stage:'post',    budget:180, irr:30 },
  { id:'P06', title:'Проект Foxtrot', stage:'pre',     budget:420, irr:22 },
  { id:'P07', title:'Проект Gamma',   stage:'pre',     budget:220, irr:18 },
];

function M2Builder() {
  // Старт: все в rail, колонки ПУСТЫЕ
  const [placement, setPlacement] = useState({
    rail: PIPELINE_SEED.map(p => p.id),
    pre: [], prod: [], post: [], release: []
  });
  const [dragId, setDragId] = useState(null);

  // Compute derived
  const stagedIds = ['pre','prod','post','release'].flatMap(c => placement[c]);
  const staged = stagedIds.map(id => PIPELINE_SEED.find(p => p.id === id));
  const totalBudget = staged.reduce((a,p) => a + p.budget, 0);
  const weightedIRR = totalBudget > 0
    ? staged.reduce((a,p) => a + p.irr * p.budget, 0) / totalBudget
    : 0;
  
  const onDrop = (col) => (e) => {
    e.preventDefault();
    if (!dragId) return;
    const next = Object.fromEntries(
      Object.entries(placement).map(([k,v]) => [k, v.filter(id => id !== dragId)])
    );
    next[col] = [...next[col], dragId];
    setPlacement(next);
    setDragId(null);
  };

  const resetToCanon = () => setPlacement({
    rail: [],
    pre: PIPELINE_SEED.filter(p => p.stage==='pre').map(p => p.id),
    prod: PIPELINE_SEED.filter(p => p.stage==='prod').map(p => p.id),
    post: PIPELINE_SEED.filter(p => p.stage==='post').map(p => p.id),
    release: PIPELINE_SEED.filter(p => p.stage==='release').map(p => p.id),
  });

  const overload = (col) => placement[col].length > 3;
  
  // render rail (if rail.length > 0) + 4 columns + stats
}
```

**Поведение:**
- Старт: все 7 в rail, колонки пусто, IRR 0%, total 0
- Drag из rail → колонка: в колонке появляется, из rail уходит
- Drag между колонками: перемещается
- Drag из колонки → rail: возвращается (уменьшает IRR)
- Reset to Canon: все 7 разлетаются по Canon-стадиям (rail пусто)
- Warning «Перегрузка стадии» если > 3 в одной колонке
- Empty state в пустых колонках: «Перетащите сюда»

**Анимация:** при drop — FLIP (getBoundingClientRect до/после + transform transition).
**Reset to Canon** — с анимацией: одновременный transfer всех карточек.
